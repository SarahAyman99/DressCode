# DressCode
Project Overview This project is a modern, responsive web application designed to present the dress code guidelines for Innovation University. Moving away from traditional, text-heavy policy documents, the primary goal was to create a highly visual, user-friendly interface that clearly communicates the university's attire policies.
By utilizing interactive UI elements—such as a context-aware "Dress Checker" tool, visual tags (Wear/Avoid), and an interactive image gallery—the application ensures students can instantly understand what to wear for any specific occasion, reducing ambiguity and improving compliance. The design prioritizes accessibility, dark-mode aesthetics, and a seamless user experience across all devices.

Technologies Used HTML5: Used for structuring the semantic content and layout of the web application. CSS3 & Tailwind CSS: Used for styling, layout grid, responsive design, and creating modern UI animations (transitions, hover effects, gradients). Tailwind CSS was integrated via CDN for utility-first styling. Vanilla JavaScript (ES6): Used to power all interactive features, including the Dress Checker logic, dynamic tab switching, the image lightbox, intersection observer animations, and the custom cursor effect. Iconify: Used via CDN to render high-quality SVG icons (Lucide icon set) for visual cues and navigation. Google Fonts: Integrated 'Inter' for body text and 'Playfair Display' for headings to ensure clean typography. Hosting & Deployment: GitHub Pages for live hosting. Version Control: Git & GitHub for source code management and tracking.
